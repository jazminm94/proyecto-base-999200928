## Descripción

Este proyecto consiste en la maquetación de una interfaz gráfica inspirada en WhatsApp.

Se reprodujo la estructura general de la aplicación, incluyendo:

* Un sidebar con la lista de chats
* Un panel principal con la conversación activa

El objetivo del proyecto es practicar maquetación y estilos visuales utilizando HTML y CSS.

### Tecnologías utilizadas

* HTML
* CSS

## Estructura del proyecto

/index.html
/styles/main.css
/assets/images

## Cómo visualizar el proyecto

1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en cualquier navegador web.

## Decisiones de diseño y consideraciones

* Se utilizó una paleta basada en todos azules y celestes para lograr una apariencia mas formal y moderna.
* Se utilizó una tipografía importada desde Google Fonts ('Roboto') para acercarnos mas al aspecto original de WhatsApp.
* La distribución general se realizó utilizando grid para la estructura principal y flexbox para la alineación de elementos internos.
* Los estilos fueron organizados por secciones para facilitar la lectura y el mantenimiento del código.
* El diseño es responsive, ocultando el sidebar en pantallas pequeñas mediante media queries.
* Las imágenes de perfil utilizadas corresponden a avatares genéricos y no a personas reales, con el objetivo de no usar datos personales.
